# Method Choice

## 1. One-Way Analysis of Variance (one-way ANOVA)
• A one-way ANOVA was used to test whether mean height differs across the three levels of milk drinking frequency. 
• This method is appropriate because the independent variable (NoMilkDrinking) is categorical with three groups, and the dependent  
  variable (HowTallAreYouWithoutShoesInMeters) is continuous. 
• ANOVA is used instead of multiple t-tests to control the Type I error rate.

## 2. Post-hoc Analysis (Tukey HSD)

Since the ANOVA was significant, a Tukey HSD post hoc test was conducted to identify which groups differ from each other. Tukey HSD controls the family-wise error rate by performing all pairwise comparisons among the three levels of milk drinking frequency.
